# AI Study Assistant

This project is an AI-powered study assistant that helps students
summarize notes, answer questions, and generate quizzes using
Large Language Models.

## Features

- Note summarization
- Question answering
- Quiz generation
- Automation using n8n

## Technologies

- OpenAI API
- Python (Flask)
- n8n
- HTML / JavaScript
